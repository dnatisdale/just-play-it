Playlist Guide

Master your music library offline.

Building a playlist in JUST PLAY IT. is designed to be flexible. The "Current Playlist" is your active workspace—whatever you see there will be saved when you hit the SAVE button.

1. Fresh Start (New Local Playlist)

Best for: When you just downloaded new albums or tracks to your device.

  • Clear the deck: Tap Edit in the "Current Playlist" section and select Clear All to start fresh.
  • Add Tracks: Scroll to the ADD AUDIO. section. Click Pick File(s) or Pick Folder to select your music.
  • Review: Reorder songs using the drag handles (in Edit mode).
  • Save: In the SAVED PLAYLISTS. block, type a name and click SAVE.

2. Build from the Device Library

Best for: Creating a "Best Of" list using songs already stored on your device.

  • Open Library: Tap DEVICE LIBRARY. to expand the section.
  • Select Songs: Tap the circular Selector Buttons on the right of each track.
  • Add to Workspace: Click the orange Add Tracks to Playlist button.
  • Save: Name it and hit SAVE in the playlists block.

3. The "Mix-Tape" (Music + Streams)

Best for: Mixing your local files with high-quality web streams.

  • Load Base: Add some local files or load an existing list.
  • Add Stream: Paste a direct audio URL (e.g., a `.mp3` link) into the URL box and click ADD.
  • Finalize: Enter a new name and click SAVE.
Pro Tip To update a playlist, load it first, add your new songs, then SAVE using the EXACT SAME NAME to overwrite it.

4. Managing Your Playlists

  • Load: Select a playlist, set Action to Load, and click GO.
  • Rename: Select a playlist, set Action to Rename, click GO, and enter the new name.
  • Delete: Select a playlist, set Action to Delete, and click GO to remove it.
